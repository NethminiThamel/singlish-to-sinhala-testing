1.Singlish to Sinhala translation testing

Automated Testing for Singlish to Sinhala Translator using Playwright

This project automates functional and UI testing of a Singlish to Sinhala translator

Tests verify translation accuracy, UI behavior, and error handling

Playwright is used as the automation framework

2.Technologies Used

Node.js

Playwright

JavaScript

Chromium, Firefox, WebKit browsers

VS Code

3.Project Structure

tests/Singlish_to_Sinhala.spec.js – Contains Playwright test cases

playwright.config.js – Playwright configuration

est-results/ – Test execution reports

package.json – Project dependencies

4.Test Scope

Singlish to Sinhala translation accuracy

Positive functional test cases

Negative functional test cases

UI behavior (real-time translation, rendering)

5.How to Set Up the Project

Clone the repository

Open project in VS Code

Install dependencies using npm install

Install Playwright browsers

6.How to Run Tests

Tests can be executed using Playwright test runner

Tests run on multiple browsers

HTML report is generated after execution

7.Test Results & Reporting

Pass/Fail status shown in terminal

HTML report availability
